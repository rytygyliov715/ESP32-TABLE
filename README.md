## 功能设计

舵机角度控制
servo_angle.cpp
描述：
模式1，舵机初始状态，舵机角度为0
模式2，舵机角度为30度
模式3，舵机角度为60度
模式4，舵机角度为90度
模式2，3，4相互变化时，需要先将舵机角度设置为0度（设置为模式1），再设置为30度，60度，90度
